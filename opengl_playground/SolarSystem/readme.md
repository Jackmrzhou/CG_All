## 太阳系

### 运行

运行Release文件夹中的 SolarSystem.exe

### 完成内容

1. 太阳自转。
2. 地球自转，并绕太阳进行公转。
3. 月球绕地球公转。
4. 3D 模型导入。
5. 实现以太阳为平行光源的漫反射与环境光源渲染。
6. 实现观察者的移动，支持前后左右移动，并支持根据鼠标运动实现观察方向的变化。

### 操作指南

1. W键为往观察方向前进，S键为往观察方向后退。
2. A键为往左移动，D键为往右移动。
3. 鼠标移动即可改变观察方向。

### 自行构建指南

1. 下载安装 glad 库、glut 库、glm 库、assimp库。
2. 在x86环境下构建。
3. 确保opengl版本在3.3以上。

### 其他

由于我下载的glut库缺乏对鼠标滚轮的支持，因此未能实现通过鼠标滚轮来实现缩放功能。
